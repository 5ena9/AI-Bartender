# 프리미엄 주류 경험 추천 서비스

## 이 버전에서 하는 일

사용자가 다음 항목을 선택하면 규칙 기반으로 위스키·와인·샴페인을 추천합니다.

- 함께 마시는 사람
- 분위기
- 도수
- 맛과 향
- 예산

추천 결과에는 주류의 특징과 선택 이유, 페어링, 음악, 처음 마시는 방법이 함께 표시됩니다.

## 실행 방법

### 1. Python 설치

Python 공식 홈페이지에서 Python 3.11 이상을 설치합니다. 설치 화면에서 `Add Python to PATH`를 체크합니다.

### 2. 터미널 열기

이 폴더에서 PowerShell을 열고 아래 명령어를 입력합니다.

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
streamlit run app.py
```

실행하면 웹 브라우저에서 주소가 열립니다. 보통 `http://localhost:8501`입니다.

## 초보자가 수정할 곳

`app.py` 위쪽의 다음 세 리스트부터 수정하면 됩니다.

- `DRINKS`: 주류 이름, 맛, 가격, 설명
- `PAIRINGS`: 음식 이름과 주류 종류별 조합
- `MUSIC`: 음악 이름, 분위기, 추천 이유

## 추천 원리

사용자가 고른 항목과 주류의 태그가 일치할수록 점수를 높입니다.

- 함께 마시는 사람 일치: 3점
- 분위기 일치: 3점
- 도수 일치: 2점
- 맛·향 하나 일치: 2점
- 예산 일치: 2점

점수가 높은 주류 3개를 보여줍니다. 이후에는 실제 사용자의 클릭·평점 데이터를 저장해 개인화 추천으로 발전시킬 수 있습니다.
