# AI Bartender 팀 프로젝트 기획서

## 1. 문제 정의
사용자가 원하는 분위기와 주류 취향을 입력하면 적절한 칵테일을 추천받기 어렵다.

## 2. 목표 사용자
칵테일을 잘 모르지만 자신의 기분과 취향에 맞는 술을 찾고 싶은 사용자

## 3. 핵심 기능
- 자연어 취향 입력
- 분위기·도수·주류 타입 분석
- 칵테일 추천
- 추천 이유와 주류 정보 제공
- 추천 결과 저장

## 4. 사용자 흐름
취향 입력 → AI 분석 → 칵테일 추천 → 상세 정보 확인 → 저장

## 5. 기술 스택
React, TypeScript, Vite, Tailwind CSS, Vercel, OpenAI API, Supabase

## 6. 배포 URL
https://ai-bartender-green.vercel.app

## 7. MVP 완료 기준
- 자연어 입력 가능
- 추천 결과 출력
- 배포 환경에서 정상 동작
- 추천 결과 저장 가능
